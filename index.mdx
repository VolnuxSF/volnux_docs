# Welcome to Volnux

Volnux is an open-source workflow orchestration platform designed for event-driven architectures. It simplifies the creation, management, and visualization of complex workflows and event pipelines.

## Key Features

- **Event-Driven Workflows** - Build workflows triggered by events in your system
- **Visual Workflow Builder** - Design and visualize workflows with an intuitive UI
- **Real-Time Processing** - Process events in real-time with multi-step workflows
- **Extensible Plugin System** - Extend Volnux with custom integrations and actions
- **Event Pipelines** - Create sophisticated event processing pipelines

## Getting Started

New to Volnux? Start with our [overview guide](/getting-started/overview) to understand the core concepts, then follow the [installation guide](/getting-started/installation) to set up Volnux in your environment.

## Core Concepts

Before diving into implementation, familiarize yourself with these fundamental Volnux concepts:

- **Workflows** - Define sequences of actions triggered by events
- **Events** - Signals that trigger workflow execution
- **Triggers** - Conditions that initiate workflows
- **Actions** - Individual steps within a workflow
- **Event Pipelines** - Chains of event processing steps
