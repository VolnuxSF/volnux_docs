# Installation Guide

Get Volnux up and running in minutes.

## Prerequisites

Before installing Volnux, make sure you have:

- **Node.js** version 18 or higher
- **npm** or **yarn** package manager
- **Git** for cloning the repository
- 2GB RAM minimum (4GB recommended)

## Installation Methods

### Option 1: From GitHub (Recommended)

Clone the official Volnux repository:

```bash
git clone https://github.com/VolnuxSF/volnux.git
cd volnux
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Your Volnux instance will be available at `http://localhost:3000`

### Option 2: Using Docker

If you prefer Docker, you can run Volnux in a container:

```bash
docker pull volnux/volnux:latest
docker run -p 3000:3000 volnux/volnux:latest
```

### Option 3: From npm (When Available)

```bash
npm install -g volnux
volnux start
```

## Verification

After installation, verify that Volnux is running correctly:

1. Open your browser and navigate to `http://localhost:3000`
2. You should see the Volnux dashboard
3. Check the terminal for any error messages

If you see the dashboard, congratulations! Volnux is successfully installed.

## Configuration

### Environment Variables

Create a `.env` file in the root directory:

```bash
NODE_ENV=development
PORT=3000
DATABASE_URL=postgresql://user:password@localhost:5432/volnux
LOG_LEVEL=info
```

### Database Setup

If using PostgreSQL, initialize the database:

```bash
npm run db:migrate
npm run db:seed
```

## Troubleshooting

### Port Already in Use

If port 3000 is already in use:

```bash
npm run dev -- --port 3001
```

### Module Not Found

Ensure all dependencies are installed:

```bash
npm install
npm audit fix
```

### Database Connection Error

Verify your database credentials in `.env` and ensure the database server is running:

```bash
psql -U postgres -d volnux -c "SELECT 1"
```
